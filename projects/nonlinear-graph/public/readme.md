# 非線形特性テーブル・グラフ表示

- 定義されている非線形特性を表形式で一覧表示します。
- 非線形特性テーブルで選択した複数のデータをグラフで表示し、比較することができます。
- EXCEL で作成した非線形特性データを非線形特性テーブルに貼り付け、CIVIL NX 本体に一括で入力することができます。

# 注意事項

- [スリップ型/圧縮のみ]と[マルチリニア-弾性]の場合、P,D(A)データを変える前に[履歴タイプ]と[軸対称]のデータを変えてください。
